Definition

The map() method is a JavaScript Higher-Order Function that creates a new array by applying a function to every element of the original array.

Easy Definition:
map() is used to transform or modify each element of an array and returns a new array.

Syntax

array.map((currentValue, index, array) => {
    // return modified value
});

Promise

A Promise is an object that represents the eventual success or failure of an asynchronous operation.

				or 

A Promise is an object that represents the future result of an asynchronous operation.

Fetch

The Fetch API is a JavaScript API used to send HTTP requests to a server. It returns a Promise, which can be handled using .then() and .catch().

Async

The async keyword declares an asynchronous function that always returns a Promise.

Await

The await keyword pauses the execution of an async function until a Promise is resolved or rejected.





const orderPizza = new Promise((resolve, reject) => {

    let pizzaReady = true;

    if (pizzaReady) {
        resolve("🍕 Pizza Delivered Successfully");
    } else {
        reject("❌ Sorry! Pizza is not available");
    }

});

orderPizza
.then((message) => {
    console.log(message);
})
.catch((error) => {
    console.log(error);
});


Promise = "I will do it later."
async = "This function works asynchronously."

introduction react :

React is a popular JavaScript library used for building fast and interactive user interfaces,
 especially for Single Page Applications (SPAs).
It was developed by Facebook. 
React follows a component-based architecture,
where we divide the application into small,
reusable components like a Navbar, Sidebar, Product Card, and Footer.
This makes development faster, improves code reusability,
and makes applications easier to maintain.

React uses the Virtual DOM, which updates only the changed parts of the page,
 improving performance



 1. What is State?
State is an object that stores dynamic data in a component. When the state changes, 
React automatically re-renders the component and updates the UI.

2. What is useState?
useState is a React Hook used to create and manage state in a functional component.


3.diff between props vs state
Props are used to pass data from a parent component to a child component, 
whereas state stores data that belongs to a component and can change over time.


4. What is the Virtual DOM?

The Virtual DOM is a lightweight copy of the Real DOM. 
When data changes, React first updates the Virtual DOM,
 compares it with the previous version, 
and updates only the changed elements in the Real DOM. This improves performance.



5. What is List Rendering?
List rendering means displaying multiple items using the map() method.
